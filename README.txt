NANA'S PLAN V2 — GITHUB PAGES UPLOAD

Upload the CONTENTS of this folder to the root of your GitHub repository:
- index.html
- manifest.webmanifest
- sw.js
- assets folder

Then on GitHub:
1. Open the repository.
2. Settings > Pages.
3. Under Build and deployment, choose Deploy from a branch.
4. Branch: main, folder: /(root), then Save.
5. Wait a few minutes for the site URL.

On iPhone/iPad:
Open the GitHub Pages link in Safari > Share > Add to Home Screen.

Important:
- This version is free and stores progress separately on each device.
- The selected reminder time is saved, but guaranteed background reminders require Apple Reminders or an external push service.
- This educational plan should be reviewed by her clinician or physical therapist.
